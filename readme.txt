Random Quote Generator is a simple web-app that uses an API call using jQuery to fetch a random quotation
from a wordpress blog on quotes. It then prints this quotation on the browser.

I developed this web-app as a part of my projects while taking Front-End Web Development lessons
 @ Free Code Camp (https://www.freecodecamp.com).

User Stories :
1. User can click a button to show me a new random quote.
2. User can press a button to tweet out a quote.

Tech-Stack:
1. HTML/CSS
2. Javascript
3. jQuery
4. Bootstrap


For a working demonstration, visit :- https://codepen.io/ekamwalia/pen/zwyKLW?editors=1010
